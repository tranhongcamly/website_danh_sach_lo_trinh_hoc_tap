Buổi 1 - KHÁM PHÁ THẾ GIỚI MÁY TÍNH ĐIỆN TỬ

  • Dự án Xây dựng cỗ máy tính ảo: Khám phá các linh kiện phản cứng, phần mềm và xây dựng máy tính trên nên tảng ảo

Buổi 2 - MÁY TÍNH VÀ CHƯƠNG TRÌNH MÁY TÍNH

  • Tìm hiểu chủ đề: Các công ty Công nghệ hàng đầu thể giới và giá trị niêm yết

  • Dự án Code Combat: Sử dụng ngôn ngữ lập trình để hoàn thành các thử thách Code Combat

Buổi 3 - LÀM QUEN VỚI LẬP TRÌNH PYTHON - HELLO WORLD

  • Tìm hiểu chủ đề: Lịch sử Ngôn ngữ giao tiếp
